How the AI works

Each movie is turned into a text document: title + genres + mood tags + overview. We build a TF-IDF vector space and rank by cosine similarity to your mood / free-text query, lightly blending catalog quality scores. Likes inject those films’ profiles into the query (content-based personalization). Offline self-retrieval metrics ship with the Python engine (/api/eval or python -m eval.run_eval).
