How it works

Face path: OpenCV Haar cascade finds your face; FER+ ONNX classifies expression (happiness, sadness, anger, …) and maps it to a mood for the recommender.
Rank path: TF-IDF over title + genres + mood tags + overview → cosine similarity.
UI path: Match loading bar → 4:3 liquid-glass card stack (linked-list prev/next), Netflix-style.
